# Accessibility with fonts

General rules with HTML text - ensure: 

* **color contrast ratios**
  * to meet _**Level AA**_
    * 4.5-to-1 for normal text
    * 3-to-1 for large text 
  * to meet _**Level AAA**_
    * 7-to-1 for normal text
    * 4.5-to-1 for large text
* **appropriate font sizing**
  * use `16px` as the reference point for the standard font size
  * ensure differentiation in sizes for the [heading hierarchy](accessibility-with-hierarchies.md#heading-hierarchies)
    * `<h1>` should get largest font size
    * a heading should not have a smaller font size than a lower-level heading
